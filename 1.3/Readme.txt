In 1.3 folder:

In folder Text21 and folder Imagept21, there are all users' individual data.

cutText21.R and cutImagept21.R are used for separating result from part 1.2. 
The output are separated in folder Text21 and folder Imagept21.

refineText21.R and refineImagept21.R are used to refine every individual data in folder Text21 and folder Imagept21.
The output are finalText21.csv and finalImage21.csv files.

Text21_Part1_3.R and Image21_Part1_3.R are used for produce descriptive data, histograms and boxplots.
The output data, histograms and boxplots are all put in report.
